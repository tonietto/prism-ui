# Deveras


UI theme for Semantic UI created for personal project.


## Instalation


`npm -i -S deveras`


## Usage


```
<!-- CSS -->
<link rel="stylesheet"  href="node_modules/deveras/dist/semantic.min.css" type="text/css">
<!-- JS -->
<script src="node_modules/deveras/dist/semantic.min.js"></script>
```
