# Prism UI

:construction:

Sports theme for Semantic UI.


## Instalation


`npm -i -S prism-ui`


## Usage


```
<!-- CSS -->
<link rel="stylesheet"  href="node_modules/prism-ui/dist/semantic.min.css" type="text/css">
<!-- JS -->
<script src="node_modules/prism-ui/dist/semantic.min.js"></script>
```
